## Этап 1 — Анализ рынка

### Главные тренды (май 2026)

Рынок разделился на два полюса: перенасыщенный дженерик-фитнес и растущий нишевый, человечный, физиологически грамотный контент. **83% аудитории уже распознают AI-сценарии** и немедленно листают дальше.

| Тренд | Описание |
|-------|----------|
| **Longevity fitness** | Тренинг «ради жизни, а не ради тела». Клиенты спрашивают «как прожить дольше», а не «как похудеть». #1 цель 2026 по опросу NASM |
| **Nervous system regulation** | Не «снижай стресс», а «тренируй вагальный тон». Кортизол, parasympathetic state — растёт лавинообразно |
| **Cortisol-aware training** | HIIT как враг гормонального баланса, особенно 35+. Высокая вовлечённость |
| **Cycle syncing & bio-syncing** | Синхронизация тренировок с циклом. США: мейнстрим. RU: почти пустая ниша |
| **Lazy fitness / Low-effort movement** | Контрреакция на «выложись на 100%». Высокий retention у выгоревших женщин 25-38 |
| **"Elevated casual" эстетика** | Намеренный красивый стиль + цветовой грейдинг = дифференциатор |
| **Микро-тренировки <60 сек** | Короткие, actionable, с немедленной ценностью. Save rate выше длинных видео |
| **Hybrid training** | Сила + кардио + мобилити в одном блоке |

### Что сейчас умирает

- Wall Pilates challenges — аудитория устала, обесценен AI-контентом
- 12-3-30 treadmill routine — потерял 60%+ интереса
- Generic body positivity — перенасыщен, воспринимается как нечестный
- Keto/IF-linked контент — интерес упал на 48%
- Мотивационные цитаты поверх стоков — нулевое доверие
- «5 советов для похудения» заголовки — воспринимаются как clickbait
- Transformation before/after с грубым контрастом — вызывает distrust
- HIIT каждый день для похудения — противоречит wellness-нарративу 2026

### Что начинает расти

- «Soft feminine discipline» — дисциплина без токсичности, с заботой о теле
- Hormone health контент — эстроген, прогестерон, щитовидка, кортизол
- Run clubs & community fitness
- Longevity-первый подход к питанию — средиземноморская диета, anti-inflammatory foods
- Realistic aesthetic routines — «моё настоящее утро»
- Protein first / protein tidbits — факты о протеине
- Спорт для здоровья суставов, костей, мозга
- Creator-led education без инфоцыганства

### Актуальные форматы (ближайшие 2 недели)

- **Talking reel + subtitles** — говорит в камеру, честно, без монтажа-шоу
- **POV + внутренний монолог** — зритель проживает момент вместе с автором
- **Aesthetic routine (реалистичная)** — красиво, но не постановочно
- **Carousel «сохрани это»** — чеклисты, таблицы. 8-12 слайдов. Mixed media = +2.33% engagement
- **Unpopular opinion** — высокий CTR, провоцирует комментарии
- **Micro-story (15-30 сек)** — история + урок, зритель сразу в середине

## Стратегические выводы

### Приоритеты на ближайшие 2 недели
- Запустить серию по cycle syncing — 4-6 Reels по фазам (быстрый захват RU-ниши)
- Сделать 2-3 кортизол/гормональных видео — пик роста
- Добавить 1 carousel в неделю (чеклист, таблица, FAQ)
- Начать строить нарратив «soft discipline» — фундамент личного бренда

### Что НЕ делать
- Не делать challenge-контент
- Не использовать before/after с резким контрастом
- Не делать «мотивационные» посты — 0 retention, 0 saves
- Не копировать западные тренды 1 в 1 без адаптации под RU-аудиторию

### Формула личного бренда 2026
> **Authentic person + физиологическая грамотность + soft discipline = устойчивый рост**
>
> Алгоритм наказывает AI-контент и вознаграждает живость, узнаваемость, retention. Задача — быть человеком, которому доверяют, а не экспертом, который давит.
