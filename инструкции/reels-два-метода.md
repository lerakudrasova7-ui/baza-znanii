# Два метода написания сценария для Reels

> Алгоритм Instagram · Психология удержания · Нейронаука

---

## Алгоритм Instagram 2025–2026

Адам Моссери подтвердил 3 главных сигнала ранжирования:
- **Watch Time** — время просмотра
- **Likes per Reach** — лайки на охват
- **Sends per Reach** — DM-шеры на охват

94% дистрибуции идёт через AI-рекомендации. Алгоритм анализирует кривую удержания по секундам — штрафует ранние выходы, вознаграждает плавное удержание до конца.

---

## Метод 1 — «Петля Curiosity»

**Захватить → Удержать → Зациклить**

### Суть
Строится на эффекте Зейгарник: незавершённое действие удерживает внимание сильнее завершённого. Сценарий открывает петлю в первые 3 секунды и не закрывает до конца.

### Нейронаука
- **Эффект Зейгарник (1927):** незавершённые задачи активируют дорсолатеральную префронтальную кору, удерживают рабочую память в «активном ожидании»
- **Curiosity Gap (Loewenstein, 1994):** человек испытывает дискомфорт от «информационного разрыва» и досматривает до конца

### Как работает на алгоритм

| Сигнал | Как влияет |
|--------|-----------|
| Watch Time | Плавная кривая удержания без провалов |
| Skip Rate | Хук снижает skip rate ниже 30–40% (штраф при >50%) |
| Replay | Незакрытая петля провоцирует пересмотр |
| Охват новой аудитории | Watch time — главный сигнал для unconnected reach |

### Структура сценария

| Время | Блок | Задача |
|-------|------|--------|
| 0–3 сек | **ХУК-ПЕТЛЯ** | Открыть незакрытый вопрос, парадокс или провокацию |
| 3–8 сек | **НАГНЕТАНИЕ** | Не давать ответ, строить контекст, каждые 3–4 сек новый мини-крючок |
| 8–22 сек | **НАРАСТАЮЩЕЕ НАПРЯЖЕНИЕ** | Доказательства, детали, темп — смена кадра каждые 1.5–2 сек |
| 22–28 сек | **PAYOFF** | Дать обещанный ответ — чётко, конкретно, без воды |
| 28–30 сек | **НОВАЯ ПЕТЛЯ** | Закрыть одно → открыть следующее (провоцирует пересмотр) |

### Примеры хуков
- «Я случайно нашёл способ набрать 50К за 2 недели — и почти не поверил сам себе...»
- «Это видео удалят. Посмотри пока не убрали.»
- «Все говорят снимай каждый день. Я снял 2 рилса за месяц. Вот что вышло.»

### Ошибки
- НЕ начинать с «привет, меня зовут...»
- НЕ объяснять, о чём будет видео
- Слабый payoff после сильного хука = разочарование = выход

### Когда использовать
✓ Привлечение новой холодной аудитории  
✓ Экспертные факты и неожиданные выводы  
✓ Образовательный контент с инсайтом  
✓ Короткие форматы 15–30 секунд  

✗ Продающий контент тёплой аудитории  
✗ Когда нужны DM-шеры  
✗ Эмоциональные темы с личной историей

## Метод 2 — «Зеркало Трансформации»

**Узнать себя → Почувствовать → Переслать**

### Суть
Работает через эмоциональную идентификацию. Зритель видит себя в герое сценария — активируются зеркальные нейроны и амигдала. Люди пересылают то, в чём узнали себя.

### Нейронаука
- **Зеркальные нейроны (Риззолатти, 1990-е):** мозг активирует те же зоны при наблюдении за чужим опытом, что и при собственном переживании
- **Амигдала:** эмоционально заряженная информация запоминается в 5–10 раз лучше нейтральной
- **Нарративный транспорт (Green & Brock, 2000):** погружённые в историю люди воспринимают выводы как собственные

### Как работает на алгоритм

| Сигнал | Как влияет |
|--------|-----------|
| DM-шеры | Главный сигнал для unconnected reach — люди пересылают то, в чём узнали себя |
| Сохранения | Практический инсайт внутри истории провоцирует save |
| Watch Time | Удержание через эмоцию, плавная кривая без резких выходов |
| Лайки на охват | Эмоциональный контент = лайки как рефлекс |

### Структура сценария

| Время | Блок | Задача |
|-------|------|--------|
| 0–3 сек | **ХУК — точка боли** | Назвать ситуацию, в которой зритель себя узнаёт. Начинать с «ты» или ситуации |
| 3–8 сек | **КОНФЛИКТ** | Углублять боль, быть конкретным — детали делают историю настоящей |
| 8–15 сек | **ПОВОРОТНАЯ ТОЧКА** | Один конкретный момент осознания — не список шагов, а один инсайт |
| 15–25 сек | **ТРАНСФОРМАЦИЯ** | Конкретный результат с цифрами, фактами, датами |
| 25–30 сек | **ИДЕНТИФИКАЦИЯ + CTA** | Мост между историей и зрителем, CTA вытекает из истории |

### Примеры хуков
- «Год назад я выкладывал каждый день. 0 подписчиков. 0 продаж. Хотел всё бросить.»
- «Ты когда-нибудь снимал рилс 3 часа — а он набрал 50 просмотров?»

### Ошибки
- НЕ начинать с биографии
- НЕ торопиться выходить из конфликта
- Поворот должен быть неожиданным, но логичным
- НЕ заканчивать на результате — заканчивать на зрителе

### Когда использовать
✓ Продающий контент для тёплой аудитории  
✓ Личный бренд и доверие через уязвимость  
✓ Ниши: психология, бизнес, отношения, здоровье  
✓ Форматы 25–45 секунд  

✗ Холодная аудитория без контекста  
✗ Развлекательный вирусный контент  
✗ Когда нет настоящей личной истории

## Сравнение методов

| Параметр | Петля Curiosity | Зеркало Трансформации |
|----------|----------------|----------------------|
| Главный сигнал | Watch Time | DM-шеры (Sends per Reach) |
| Психо-механизм | Эффект Зейгарник | Зеркальные нейроны + амигдала |
| Аудитория | Холодная / новая | Тёплая / лояльная |
| Длина | 15–30 сек | 25–45 сек |
| Цель | Охваты + подписчики | Продажи + пересылки |
| Тип хука | Вопрос / парадокс | Боль / узнавание |
| Метрика | Completion Rate / Replay | Saves / DM-шеры |

## Стратегия чередования

Два метода не конкурируют — закрывают разные задачи. Оптимально: **чередовать 2:1 или 3:1**.

| День | Метод | Цель |
|------|-------|------|
| Пн | Петля Curiosity | Новая аудитория → охваты, подписки |
| Ср | Петля Curiosity | Вирусный охват → replay, completion rate |
| Пт | Зеркало Трансформации | Продажи/шеры → DM, сохранения, конверсия |
